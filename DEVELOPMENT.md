Development notes

- Admin password can be configured via environment variable `VITE_ADMIN_PASSWORD`. Default: `admin123` (local dev only).
- Useful commands:
  - `npm run dev` — start vite dev server
  - `npm run dev:node` — start vite directly via node (useful on Windows)
  - `npm run build` — build production assets
  - `npm run preview` — serve built `dist` locally
  - `npm run lint` — run ESLint
